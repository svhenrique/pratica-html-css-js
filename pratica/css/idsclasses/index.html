id:
- São únicos em uma página
- No CSS são usados com #x sendo que x é o nome do id no html

class:
- classes servem essencialmente para juntar várias tags html em um mesmo escopo
- uma tag pode ser de várias classes
- usado para aplicar diferentes estilos no CSS
- no css chamamos uma classe como .x sendo x o nome da classe no html

- ids são mais fortes que as classes
- Se uma tag tiver várias classes que sobrepõem características, a a classe mais forte será a ultima da esquerda pra direita
